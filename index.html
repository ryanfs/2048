Use your arrow keys to move the tiles and get to 2048!

 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 

Coded from scratch by Ryan F. Salerno while at Dev Bootcamp. Based on the original 2048 game by Gabriele Cirulli.

Written using Underscore.js for enumerable object manipulation, jQuery for key bindings, and vanilla JavaScript for the bulk of the game logic.
